# Accounting OCR Frontend

NextJS frontend for the Accounting OCR Platform.

## Local Run

```bash
cd frontend
npm install
npm run dev -- --port 3001
```

Open `http://localhost:3001`.

Current pages:

- `/`
- `/dashboard`
- `/accounting`
- `/ai`
- `/admin`

Set the backend URL with:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8001/api/v1
```
